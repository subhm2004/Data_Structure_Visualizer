# DS Visualizer

Interactive **Data Structures & Algorithms Visualizer** — React + Vite + Three.js. 18 visualizers with step-by-step animations. Pure frontend, no backend.

## Live Demo

Deploy on [Vercel](https://vercel.com) or [Netlify](https://netlify.com) — connect this repo, build command `npm run build`, output `dist`.

## Visualizers

- **Sorting** — 9 algorithms (bars, numbers, merge/quick tree)
- **Array** — search, insert, delete
- **Trees** — Binary, AVL, Heap, Trie, Recursion
- **Linked Lists** — singly & doubly
- **Stack, Queue, Circular Queue**
- **Hash Map** — chaining & linear probing
- **Graph** — DFS/BFS, Dijkstra
- **Advanced** — Fenwick Tree, Segment Tree, 2D Prefix Sum

## Quick Start

```bash

npm run dev
```
